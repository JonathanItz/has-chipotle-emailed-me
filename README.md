# Node/Express Boilerplate

A fast and simple way to get a project started

## Installation

### Step 1:

`git clone https://github.com/JonathanItz/Node-Boilerplate.git`

or

`fork this repo`

### Step 2:

`npm i`

## What's being used?

<sup>it's simple<sup>

* EJS
* SCSS
* Node/Express (obviously)
